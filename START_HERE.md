# SafeVision AI - Start Here

This folder contains the complete SafeVision AI demo:

- `app.py` - Streamlit landing page and live dashboard in one app
- `assets/safevision_demo_video.mov` - recorded demo video shown on the landing page
- `assets/SafeVision_AI_Architecture.png` - architecture diagram used by the landing page and dashboard
- `sample_videos/` - bundled CCTV demo footage
- `models/` - YOLO model files

Run the demo:

```bash
cd /Users/manavdoshi/Documents/Codex/2026-06-22/build-a-complete-macbook-air-m1/SafeVision-AI
source .venv/bin/activate
streamlit run app.py
```

Open the landing page, then click **Launch Live Dashboard** to enter the original SafeVision operations dashboard.
